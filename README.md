# 📊 Global Business Performance Dashboard

An interactive Business Intelligence dashboard built with Power BI to analyze global sales performance using KPIs, trend analysis, and geographic insights.

## Features

- 📈 Total Revenue, Profit and Quantity KPIs
- 🌍 Sales performance by Country
- 📦 Sales analysis by Category
- 📅 Year-over-Year Sales Trend
- 🔍 Interactive slicers for Country filtering
- 📊 Business-oriented dashboard design

---

## Tools Used

- Microsoft Power BI
- Power Query
- Data Modeling
- Data Visualization
- Business Intelligence

---

## Business Questions Answered

- What drives revenue performance?
- Which countries generate the highest sales?
- Which product categories perform best?
- How has revenue changed over time?
